Default = LazyInit | Shared

Specifies the default behavior, combining lazy initialization and shared instance semantics.

This value is typically used when no specific configuration is provided, ensuring that the instance is created only when needed and is shared across consumers.

GroupShared = 4

Allows a resource to be shared in a single GraphResourceContainer.

LazyInit = 1

Resources are lazily initialized. This means the resource is not immediately available, but is guaranteed available after the graph has been built.

None = 0

No special flags set.

Shared = 2

Resources are shared and can be used by multiple effects/passes. Note: Shared resources require LazyInit to be set.
