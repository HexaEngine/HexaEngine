Represents a compiler for shader programs that can compile shader source code into shader objects.

public interface IShaderCompiler

Compile(string, ShaderMacro[], string, string, string, Shader**, out string?)

Compiles shader source code into a shader object.

bool Compile(string code, ShaderMacro[] macros, string entry, string sourceName, string profile, Shader** shader, out string? error)

Parameters

code string

The source code of the shader.

macros ShaderMacro[]

An array of shader macros to use during compilation.

entry string

The entry point for the shader.

sourceName string

The name of the shader source.

profile string

The shader profile to compile for.

shader Shader**

When the method returns true, contains a pointer to the compiled shader; otherwise, null.

error string

When the method returns false, contains the error message; otherwise, null.

Returns

bool

True if the compilation is successful; otherwise, false.
